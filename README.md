## Getting started

This is a basic VueJs, Vuetify, Vue Router and Vuex skeleton written on JavaScript.

## Features

-   Vuetify
-   Multiple environment ready (development, production).
-   Vue router
-   Vuex
-   i18n ready.
-   PWA
-   Users admin area with CRUD operations.
